# Tmux Cheat Sheet

## Daily minimum

- `Ctrl-a` ‚Äî leader
- `Ctrl-a c` ‚Äî new window
- `Ctrl-a Enter` ‚Äî split horizontal
- `Ctrl-a =` ‚Äî split vertical
- `cmd+f` / `cmd+–∞` ‚Äî persistent popup shell
- `cmd+alt+f` / `cmd+alt+–∞` ‚Äî ephemeral popup shell
- `Ctrl-a s` ‚Äî session picker
- `Ctrl-a w` ‚Äî workmux dashboard popup
- `Ctrl-a v` ‚Äî copy-mode
- `y` / `Enter` ‚Äî copy
- `q` ‚Äî quit copy-mode
- `Ctrl-a r` ‚Äî reload config
- `cmd+j` / `cmd+–æ`, `cmd+k` / `cmd+–ª` ‚Äî prev / next window

## If Russian layout

- `v` `–º`
- `y` `–Ω`
- `q` `–π`
- `r` `–∫`
- `e` `—É`
- `w` `—Ü`

- `Ctrl-b` disabled
